# PROBLEM.md

**Project:** `odd-sum` (toy example)

**Goal (verbatim).** Prove that for every positive integer $n$, the sum of the
first $n$ positive odd numbers equals $n^2$. That is,

$$1 + 3 + 5 + \cdots + (2n - 1) = n^2 \qquad \text{for all } n \ge 1.$$

This is a synthetic, elementary target used only to demonstrate the
`write-paper` pipeline end to end. It carries no special domain assumptions.
