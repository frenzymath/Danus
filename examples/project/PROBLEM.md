**Project:** `triangular-sum` (toy example — illustrative sample data, not a real run)

**Goal (verbatim).** Prove that for every positive integer $n$, the sum of the
first $n$ positive integers equals $n(n+1)/2$. That is,

$$1 + 2 + 3 + \cdots + n = \frac{n(n+1)}{2} \qquad \text{for all } n \ge 1.$$

This is a synthetic, elementary target used only to show the *shape* of a Danus
project on disk — a verbatim `PROBLEM.md` plus a small `fact_graph/`. It carries
no domain assumptions and was **not** produced by a live verified run; the two
facts under `fact_graph/facts/` are hand-authored sample data (see the note in
`fact_graph/README.md`).
